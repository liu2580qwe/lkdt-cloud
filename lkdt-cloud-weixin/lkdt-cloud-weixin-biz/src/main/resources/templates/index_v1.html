《道路交通安全法实施条例》（第五节 高速公路的特别规定）
《江苏省恶劣天气条件下高速公路交通管制工作规范》
【局部路段团雾预警规则】：以高速公路交警大队管辖路段或路段管理（分）中心为一个预警计算单元， 当遇有局部路段团雾或烧秸秆烟雾等事件导致与同路段相邻监测点的能见度值相差大于100米时（容易引起司机瞬间黑视或恐慌）， 推送“前方团雾谨慎驾驶”预警信息。团雾（烟雾）预警范围尺寸与路段摄像机视角覆盖路段范围一致。
